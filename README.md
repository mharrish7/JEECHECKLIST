# JEECHECKLIST
    Its a simple app to track jee prep progress
# Build with
    Kivy,KivyMD,a-kivymd,Buildozer,python,mysql.
    
# Features
    * The data is visually represented to make the track easier
    * Comes with pre-built chapters
    * Easy to understand UI
